# SecurePay - UPI Payment Gateway

A complete static payment page hosted on GitHub Pages for UPI payments.

## Features
- ✅ Generate UPI payment links
- ✅ QR code display
- ✅ Transaction history in browser
- ✅ Dark/light mode
- ✅ Receipt generation and download
- ✅ Mobile responsive
- ✅ Input validation
- ✅ Professional design

## Setup
1. Clone this repository
2. Replace UPI ID in `js/payment.js`
3. Deploy to GitHub Pages

## How It Works
1. Customer enters payment details
2. Generate UPI link or QR code
3. Customer pays via UPI app
4. Customer enters transaction ID
5. Generate and download receipt

## Disclaimer
This is a static payment page. Payment verification requires a backend server with bank APIs.

## Tech Stack
- HTML/CSS/JavaScript
- QR Code API
- Local Storage
- GitHub Pages
